# 🐱 QQ #047 — neco_palette

**好きな猫ちゃんからデザインを選ぼう**

> 「デザインを選ぶ前に、デザインの世界を体験する入口」

---

## 概要

猫種イラストを選ぶと、Claude API がカラーパレット・デザイン印象・配色の哲学・向いているアプリジャンルを生成します。

Vivillon Code「デザインソムリエ」構想の **Section 01 — ネコたち** として設計。
気分や好みを「好きな猫ちゃん」という直感的な問いに変換することで、デザイン初心者でも自分のスタイルを発見できます。

## 使い方

1. ブラウザで `neco-palette.html` を開く
2. Anthropic API キーを入力して「保存」
3. 好きな猫を1匹クリック
4. 「このデザインパレットを生成する」ボタンを押す
5. カラーパレット・解説・向いているジャンルが表示される

---

## 技術仕様

- 単体 HTML ファイル（依存なし・ビルドステップなし）
- Anthropic API `claude-sonnet-4-20250514` / Vision 使用
- API キーは `localStorage` に保存
- 猫イラスト画像は base64 でファイルに埋め込み済み
